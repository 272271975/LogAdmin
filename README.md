# LogAdmin
-------------
基于react和[ace-admin](http://ace.jeka.by/index.html)实现的一套管理端脚手架。
- 包含Bootstrap-table, Echart, react-router, redux, redux-form等插件，定制开发能实现管理端基本功能。
- 通过使用**Flume+Kafka+Spark Streaming+Redis+MySQL**形成数据流，为平台提供数据来源（nginx访问日志）。[系统搭建相关知识点这里](http://www.jianshu.com/p/c2e5c347fe60)
- [站点展示（该站点目前包含真实数据展示，增删改部分尚未上传）](http://121.42.36.80/log-admin/#/url/index/1)
